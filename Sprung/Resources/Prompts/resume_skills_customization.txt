## CRITICAL CONSTRAINTS

### Skill Selection
You may ONLY select skills from the Skill Bank provided in the system preamble.
Do NOT invent, fabricate, or add any skills not in the bank.
The skill bank represents verified skills from the candidate's documented experience.

### Category Generation
You ARE allowed to create new category names that:
- Match the job posting requirements
- Group related skills logically
- Are supportable by skills in the bank (don't create a category if no skills fit it)

## Job Description Context
The full job description is provided in the preamble. Use it to:
- Prioritize skills mentioned in the job requirements
- Create categories that align with the role
- Order skills by relevance to this specific job
